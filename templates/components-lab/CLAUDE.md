# CLAUDE.md — Components lab

A copy-paste palette of Tailwind components I (the designer) can pull from when building anything else. Buttons, cards, inputs, hero sections, stat rows, pull quotes.

## Working with me

- "Add a [component] to my portfolio's [section]" — pull from this lab.
- "Theme this whole lab with my brand colors" — swap stone-* and accent values throughout.
- "Add a [new component] here" — extend the lab with the new one + its snippet.

## How it's structured

Each component sits in two places:
1. **Live preview** — the actual rendered version in a `<div class="demo">`.
2. **Snippet** — the same code as a `<pre class="snippet">` for copy-paste.

When you add a new component, follow the same dual-pattern.
