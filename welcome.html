Pegasus

You're set up.

Your Mac is now a coding-ready studio. Below: the four things to do first.

1. Make your first project

In any terminal (including the one inside VS Code), type:

pegasus new portfolio my-site

This creates ~/Design-Projects/my-site with a working portfolio. VS Code opens automatically.

Other types you can make:

  • case-study-deck — slide deck for an interview
  • scroll-case-study — long-form scrollable case study page
  • landing-page — coming-soon / signup page
  • resume — printable single-page CV
  • link-in-bio — Linktree replacement on your own domain
  • illustration-gallery — masonry grid for paintings, vector art, lettering

2. Sign into your services

One command opens the sign-in pages for everything Pegasus can talk to (GitHub, Surge, Figma, Adobe, Webflow, Google):

pegasus signin

3. Talk to Claude (or OpenCode)

Inside your project folder, type:

claude       # Anthropic's agent
opencode     # open-source equivalent

Then describe what you want in plain language. Examples:

  • "Replace the hero text with my name and tagline"
  • "Pull this Figma design and rebuild it: [paste URL]"
  • "Use the job-finder skill to find me design jobs"
  • "Use the vector-workflow skill to convert these Procreate exports"

4. Ship it

When the project is ready:

pegasus deploy

Surge returns a public URL within seconds. Free for personal projects.
