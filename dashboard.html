    Anatomy of an expert prompt
    
          
    Six ingredients. Drop any one, the prompt gets weaker.
    
          
      
        
      

        
      
          prompt mastery escalation tutorial guide
          
      The composition formula
      
          
      Stack these five and you have a robust prompt.
      
          
      [ Skill or template ]
  + [ Pointed input    ]   ← real path / URL / id
  + [ Tight constraint ]   ← stack, style, perf budget
  + [ Acceptance test  ]   ← measurable done state
  + [ Loop hook        ]   ← what to ask back
= Robust result
      
          
      The skill names a tool. The inputs feed it. The constraints narrow it. The acceptance test ends the back-and-forth. The loop turns one prompt into a workflow.
